# MMO Social Game Server API

Backend API for a social MMO platform focused on:

- Player-driven in-game economy
- Secure player-to-player trading
- Social game systems
- LiveOps and admin tooling support

## Tech Stack

- Node.js
- Fastify
- TypeScript
- MongoDB (planned)

This project is designed to simulate a real-world game server backend
used by modern cross-platform social MMO games.
